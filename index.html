CUHK-loopbook

目的

幫中大倒米

用途

半自動續借中大圖書

功能

安裝後輸入你ge中大圖書館名稱&密碼(唔好用中文全形鍵盤輸入)， 以後只要間中在7-9am或7-9pm連上internet就OK(唔使開此app), 有書在48hr內過期,手機自然會有notification（大約7-9am及7-9pm）,再gum幾個掣就續借完la!

續借方法

gum上面條bar粒chrome icon

注意

如果中大屎忽痕，改左網頁格式，此app九成會fail，請聯絡我地，我地會盡快更新。

To 有意開發者

坦白講，source code裡充滿了邪惡的design pattern。對於稚嫩的程序員，此絕不可遠觀，亦不可褻玩，以免沾污思想。 此程式看似複雜，其實精華盡在LibConn.java（呢part我花左3粒鐘就寫完la，反而係android UI很麻煩）

必要技能

  • 對open source / free software的迷戀

最好懂得

  • java (其實我唔係好熟)
  • html / css / http (一點點就可以了)

運作原理

研究送到https://m.library.cuhk.edu.hk/的http封包，用jsoup模擬這些封包，並記下cookies以維持login

就係咁簡單
